Open source OLT Manager


a tools to connect to OLT and manage FRAMEs, SLOTs, PORTs, ONTs on that OLT

HUAWEI OLTs supported

other Brands can be added easily


technologies

jdk 11

spring boot

vaadin



how to build

install postgresql

mvn clean compile

mvn spring:run

enter http://localhost:8080 in you browser
